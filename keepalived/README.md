本镜像不需传递任何参数，完全基于keepalived.conf挂载到容器的方案  

所以容器在启动的时候一定要挂载自定义keepalived.conf文件到容器里/etc/keepalived/keepalived.conf  

具体模板可参考上层的conf目录下的文件  

否则会走默认的keepalived.conf，将毫无意义！
